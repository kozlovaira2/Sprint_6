# Sprint_6 - Автотесты для сервиса «Яндекс.Самокат»

## Описание проекта
Автотесты для учебного сервиса «Яндекс.Самокат» (https://qa-scooter.praktikum-services.ru).  
Проект покрывает ключевую функциональность сервиса: проверку FAQ, оформление заказа и редиректы.

## Технологии
- Python 3.9+
- Selenium WebDriver
- pytest
- Allure
- Firefox

## Установка и запуск
pip install -r requirements.txt
pytest tests/ -v --alluredir=allure_results
allure serve allure_results

## Что проверяется

test_faq.py	- 8 вопросов в разделе «Вопросы о важном» — при клике открывается правильный текст ответа. 
test_order.py - Позитивный сценарий заказа самоката (2 набора данных, 2 точки входа — верхняя и нижняя кнопки). 
test_redirects.py - Редирект на главную страницу при клике на логотип «Самоката» и на Дзен при клике на логотип «Яндекса». 

## Структура проекта
pages/ — Page Object Model (базовая страница, главная страница, страница заказа, страница Дзена). 
tests/ — тесты. 
conftest.py — фикстуры. 
data.py — тестовые данные. 
allure_results/ — результаты тестов для Allure-отчёта.
